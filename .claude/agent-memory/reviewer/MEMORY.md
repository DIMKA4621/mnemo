# Reviewer agent memory

Особистий скоуп памʼяті агента-рев'ювера.

## Review checklist
- Міграції backward-compatible? Drop колонки — окремим релізом?
- Є тест на баг-фікс (регресійний)?
- Жодних секретів/PII у логах і в комітах?
- Курсорна пагінація, не offset, на нових списках?

## Things I always flag
- Довгі транзакції в Core Service — блокують пул конектів.
- `CREATE INDEX` без `CONCURRENTLY` на проді — відхиляю одразу.
- Кеш без TTL або без явної інвалідації на запис.
