# Developer agent memory

Особистий скоуп памʼяті агента-девелопера.

## Coding conventions
- Python: PEP 8, type hints обовʼязкові, докстрінги англійською.
- Тримаємо функції короткими; побічні ефекти ізолюємо в окремий шар.

## Known pitfalls
- Не робити довгих транзакцій у Core Service — блокують пул конектів.
- Redis job payloads мають бути ідемпотентними: воркер може повторити
  задачу при падінні.
